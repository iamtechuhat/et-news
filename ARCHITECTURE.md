## Data Flow

```
1. NEWS FEED FLOW
   User Opens App вҶ’ useNewsData hook вҶ’ fetch RSS2JSON API вҶ’ Parse & Categorize вҶ’ Display NewsCards

2. TRANSLATION FLOW
   User clicks "аӨ№аӨҝаӨӮаӨҰаҘҖ аӨ®аҘҮаӨӮ аӨӘаӨўаӨјаҘҮаӨӮ" вҶ’ useGemini.translateToHindi() вҶ’ Gemini API вҶ’ Show Hindi text below card

3. BRIEFING FLOW
   User clicks "Today's Briefing" вҶ’ Take top 5 headlines вҶ’ useGemini.generateBriefing() вҶ’ Show in BriefingModal

4. STORY ARC FLOW
   User clicks "Track Story" вҶ’ useGemini.getStoryArc() вҶ’ Gemini returns JSON вҶ’ Display in StoryArc panel

5. PERSONALIZATION FLOW
   First Visit вҶ’ Onboarding Overlay вҶ’ User selects categories вҶ’ Save to localStorage вҶ’ Filter feed
```

## Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| Client-side Gemini calls | Simplifies architecture; no backend needed for hackathon demo |
| RSS2JSON proxy | Bypasses CORS restrictions on ET RSS feed |
| localStorage for prefs | No auth needed; instant personalization |
| Keyword-based categorization | Works offline without ML; fast and deterministic |
| React 18 + Vite | Fast DX, instant HMR, optimized production builds |
| TailwindCSS | Rapid UI development with consistent design tokens |
